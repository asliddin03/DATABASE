# Работа №2 по базам данных и программированию

## Модификация модели библиотеки

Этот проект представляет собой реализацию бизнес-логики библиотеки на PostgreSQL версии 13 и выше, включая DML операции, представления, хранимые процедуры и систему логирования.

## Структура проекта

### Основные компоненты:

1. **DML операции** - сценарии CRUD для основных сущностей
2. **Бизнес-процессы** - выдача, возврат, бронирование книг
3. **Представления** - для аналитики и отчетности
4. **Хранимые процедуры** - бизнес-логика
5. **Система логирования** - автоматическое отслеживание изменений

## Выполненные задания

### 1-5. CRUD операции
- Добавление/удаление/редактирование авторов
- Добавление/удаление/редактирование издательств
- Добавление/удаление/редактирование информации о книгах
- Добавление/удаление/редактирование информации о читателях
- Добавление/удаление/редактирование информации об экземплярах книг

### 6-7. Основные бизнес-процессы
- Выдача экземпляра книги читателю
- Возврат выданного экземпляра книги

### 8-9. Аналитические представления
- Информация о всех выданных книгах
- Информация о просроченных выдачах

### 10, 13. Бизнес-правила
- Запрет выдачи при наличии просрочек у читателя
- Запрет выдачи забронированных экземпляров

### 11-12. Бронирование
- Бронирование экземпляра на 3 календарных дня
- Отмена бронирования

### 14-16. Дополнительный функционал
- Поиск местоположений книг с сортировкой по состоянию
- Статистика доступных экземпляров
- Книги с просрочкой более 1 года

### 17-18. Система логирования
- Таблица логов для отслеживания изменений
- Автоматические триггеры для DML операций
